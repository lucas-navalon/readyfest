# Frontend React do CRUD

Interface simples em React para consumir o backend de usuarios.

## Como rodar

1. Instale as dependencias do frontend:

```bash
cd frontend
npm install
```

2. Inicie o backend em outra aba do terminal:

```bash
cd backend
npm run dev
```

3. Rode o frontend:

```bash
cd frontend
npm run dev
```

O Vite sobe em `http://localhost:5173` e faz proxy das requisicoes `/users` para `http://localhost:3000`.
